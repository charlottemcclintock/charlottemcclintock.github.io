Visualizing gender differences in intergenerational mobility

GenSquared: Mothers, Fathers, Daughters, Sons

GenSquared is an interactive web app using R, plotly, Shiny, and hosted through shinyapps.io. Users can filter by continent, select countries, sort by generational and gender differences, and watch the change through the decades.

The project is just getting started, check back later for more!
